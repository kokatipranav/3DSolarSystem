Solar System Orrery This is a solar system orrery based on mathematical formulas and the NASA Near Earth Comets API. The planets are displayed by using their heliocentric coordinates (x, y, z). The animation is created by moving frames of plotted planets. To calculate the planet positions, the formulas from the excellent Astronomy Answers website was used: https://aa.quae.nl/en/reken/hemelpositie.html.
